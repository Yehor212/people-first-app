## 🚨 Проблема

### Что говорят пользователи:
> "Слишком много всего на главной странице"
> "Не понятно с чего начать"
> "Перегружает"

### Психологическая причина:
**Закон Хика (Hick's Law):** Время принятия решения увеличивается с количеством вариантов.

- 2 варианта → решение за 1 сек
- 5 вариантов → решение за 3 сек
- 10+ вариантов → **паралич выбора** (особенно при ADHD)

### Техническая причина:
Главная страница показывает **10+ секций одновременно**, требуя 3-4 экрана прокрутки.

## 🎨 Новая концепция: "Focus First"

### Философия
> **"Одно действие за раз. Остальное — когда понадобится."**

### Ключевые принципы:

1. **Один Primary CTA** — система определяет что важно СЕЙЧАС
2. **Прогрессивное раскрытие** — детали по запросу
3. **Минимум прокрутки** — всё важное на одном экране
4. **Celebration каждого шага** — микро-награды за действия

## 🧠 ADHD-адаптация

### Проблемы ADHD и решения:

| Симптом | Как проявляется | Решение в дизайне |
|---------|-----------------|-------------------|
| **Paralysis of choice** | "Не знаю с чего начать" | Один Primary CTA |
| **Object permanence** | Забывают про streak | Streak всегда видимый |
| **Time blindness** | Не чувствуют время | "До конца дня: 4ч" |
| **Dopamine seeking** | Нужны награды | Micro-celebrations |
| **Executive dysfunction** | Сложно планировать | Авто-выбор следующего |
| **Overwhelm** | Паника от количества | Скрытые детали |

### Специальные функции:

#### 1. Smart Next Action
```typescript
function getNextAction(state: AppState): PrimaryAction {
  // Утро: начинаем с настроения
  if (ismorning() && !state.moodLoggedToday) {
    return { type: 'mood', priority: 'high' };
  }

  // После настроения: привычки
  if (state.habitsRemaining > 0) {
    return { type: 'habits', priority: 'high' };
  }

  // После привычек: фокус (если ещё не было)
  if (state.focusSessionsToday === 0) {
    return { type: 'focus', priority: 'medium' };
  }

  // Вечером: журнал
  if (isEvening() && !state.gratitudeToday) {
    return { type: 'gratitude', priority: 'medium' };
  }

  // Всё сделано!
  return { type: 'celebration', priority: 'low' };
}
```

## 👥 Приоритеты для обычных пользователей

### Что важно обычным пользователям:

1. **Быстрый доступ** — открыл → сделал → закрыл
2. **Понятная структура** — что где находится
3. **Прогресс виден** — streak, завершённые привычки
4. **Не надо думать** — приложение подсказывает

### Отличия от ADHD-версии:

| Функция | ADHD | Обычные |
|---------|------|---------|
| Primary CTA | Критично | Полезно |
| Time awareness | Критично | Опционально |
| Ultra Simple | Важно | Не нужно |
| Detailed view | Скрыть | Можно показать |
| Gamification | Умеренно | Полноценно |
