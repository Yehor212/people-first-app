# Чеклист релиза

Используй перед публикацией в сторах или на вебе.

## Версии
- Обновить `package.json` version.
- Обновить версию в интерфейсе (если показана).
- Обновить Android `versionName` и `versionCode`.
- Добавить git‑тег релиза.

## Сборка
- `npm install`
- `npm run build`
- `npx cap sync`

## Web/PWA
- Проверить обновление service worker.
- Проверить иконки манифеста и `start_url`.
- Прогнать Lighthouse PWA.

## iOS
- Сборка в Xcode.
- Проверить запрос разрешений на уведомления.
- Проверить deep link и запуск из фона.
- Archive и загрузка в App Store Connect.

## Android
- Собрать подписанный AAB.
- Проверить запрос разрешений на уведомления (Android 13+).
- Проверить на физическом устройстве.
- Загрузить в Play Console.

## Store listing
- Название, подзаголовок, описание.
- Скриншоты нужных размеров.
- Ссылка на политику конфиденциальности.
- Email поддержки и контактный URL.

## Финальный QA
- Пройти `docs/SMOKE_CHECKLIST_RU.md`.
- Проверить чистую установку (без кэша).
- Проверить обновление с предыдущей версии.
