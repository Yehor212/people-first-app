Politique de confidentialite

Date d'entree en vigueur: 2026-01-25

Resume

  • Vos donnees sont stockees sur votre appareil par defaut.
  • Nous ne vendons pas vos donnees.
  • Si la synchronisation est activee, les donnees sont stockees dans votre compte.

Donnees que nous stockons

ZenFlow stocke les entrees d'humeur, les habitudes, les sessions de focus et la gratitude sur votre appareil. Les donnees restent locales sauf si vous activez la synchronisation cloud.

Synchronisation cloud

Si vous vous connectez et activez la synchronisation, les donnees sont stockees dans votre compte. Cela sert uniquement a synchroniser entre appareils.

Notifications

Si vous activez les rappels, l'app planifie des notifications locales sur votre appareil. Les notifications push ne sont pas utilisees pour le moment.

Analytique

L'analytique est optionnelle et peut etre desactivee dans les reglages.

Health Connect (Android)

Si vous activez l'integration Health Connect sur Android, ZenFlow peut acceder a:

  • Lecture: Donnees de sommeil, nombre de pas - pour fournir des informations de bien-etre
  • Ecriture: Sessions de pleine conscience - pour enregistrer vos sessions de minuteur de concentration

Les donnees de sante sont traitees localement sur votre appareil. Nous ne partageons pas les donnees de sante avec des tiers.

Exigence d'age

ZenFlow est concu pour les utilisateurs de 13 ans et plus. Si vous avez moins de 13 ans, veuillez ne pas utiliser cette application sans le consentement de vos parents.
