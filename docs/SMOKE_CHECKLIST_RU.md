# Смоук‑чеклист

Пройти перед публикацией релиза.

## Подготовка
- `npm install`
- `npm run build`
- `npx cap sync`

## Основные потоки
- Онбординг проходит до конца и задает напоминания при включении.
- Главная вкладка открывается без ошибок при первом запуске.
- Вкладка статистики корректно показывает данные при пустых и заполненных списках.
- Настройки: изменение имени сохраняется после перезагрузки.

## Напоминания
- Включить напоминания и проверить запрос разрешения на устройстве.
- Убедиться, что напоминания не планируются в тихие часы.
- Убедиться, что карточки напоминаний появляются в приложении в нужное время.

## Безопасность данных
- Экспортировать данные и импортировать в режиме merge (без ошибок).
- Импортировать в режиме replace (данные заменяются корректно).
- Сброс данных очищает mood/habits/focus/gratitude.

## Offline/PWA
- Появляется приглашение установки PWA (где поддерживается).
- Приложение стартует офлайн, основные экраны рендерятся.
- Обновление service worker не ломает запуск.

## Аккаунт/синхронизация (если настроено)
- Вход по magic link работает, синхронизация завершается.
- Выход из аккаунта очищает сессию без ошибок.
