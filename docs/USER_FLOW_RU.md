# Логика пользователя (RU)

## Первый запуск
- Выбирает язык и цель.
- Добавляет 1-2 привычки.
- Включает напоминания (локальные на устройстве).
- При желании включает push и входит в аккаунт для синхронизации.

## Ежедневный цикл
- Отмечает настроение (30 секунд).
- Выполняет привычки и отмечает прогресс.
- Запускает фокус-сессию.
- В конце дня видит итог и мягкий нудж.

## Напоминания
- Локальные: работают в нативном приложении (Android/iOS).
- В приложении: всплывающие напоминания, пока приложение открыто.
- Push: опционально, работают после включения и входа в аккаунт.

## Синхронизация
- Вход через Google/Email включает облачный бэкап.
- При входе на новом устройстве данные подтягиваются.
- Можно вручную синхронизировать из настроек.

## Безопасность и приватность
- PIN/биометрия по желанию.
- Аналитика отключаема.
- Удаление аккаунта доступно в настройках.
